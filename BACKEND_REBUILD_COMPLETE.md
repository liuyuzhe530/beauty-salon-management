### 6️⃣ **数据库模型** ✅

**User表**
- id (UUID, 主键)
- username (string, 唯一)
- email (string, 唯一)
- password (string, 加密)
- role (enum: admin, staff, customer)
- isActive (boolean)

**Customer表**
- id (UUID, 主键)
- userId (UUID, 外键)
- firstName, lastName (string)
- phone (string, 唯一)
- email (string)
- address, city (string)
- notes (text)

**Staff表**
- id (UUID, 主键)
- userId (UUID, 外键)
- firstName, lastName (string)
- specialization (string)
- phone (string)
- email (string)
- isAvailable (boolean)

**Appointment表**
- id (UUID, 主键)
- customerId (UUID, 外键)
- staffId (UUID, 外键)
- service (string)
- appointmentDate (datetime)
- duration (number, 分钟)
- status (enum: pending, confirmed, completed, cancelled)
- notes (text)
- price (decimal)

**Product表**
- id (UUID, 主键)
- name (string)
- description (text)
- price (decimal)
- category (string)
- stock (number)
- isActive (boolean)
