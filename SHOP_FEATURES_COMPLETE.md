## 2️⃣ 智能比价功能详解

### 核心功能

```
✅ 价格三维对标
   ├─ 我们的价格 (当前定价)
   ├─ 市场均价 (平均市价)
   ├─ 竞争对手价 (最低竞争价)
   └─ 自动计算价格差异

✅ 优先级智能判断
   ├─ 🔴 高优先 (差价 > 15%) → 立即降价
   ├─ 🟡 中优先 (差价 5-15%) → 小幅调整
   └─ 🟢 低优先 (差价 < 5%) → 保持价格

✅ 智能定价建议
   ├─ 基于市场数据的建议
   ├─ 预期销量提升预估
   ├─ 利润空间评估
   └─ 竞争力分析

✅ 一键操作
   ├─ 点击"降价建议"
   ├─ 直接导航到编辑
   └─ 快速应用新价格
```

### UI展示

```
智能比价标签页包含:
├─ 头部说明
│  ├─ 分析标题
│  ├─ 说明文字
│  └─ 渐变背景设计
├─ 产品对标卡片 (5个)
│  ├─ 产品名称
│  ├─ 定价建议文字
│  ├─ 优先级标签
│  ├─ 价格三列对比
│  ├─ 价格差异显示
│  └─ 操作按钮
└─ 颜色编码
   ├─ 红色 (高优先)
   ├─ 黄色 (中优先)
   └─ 绿色 (低优先)
```

### 分析示例

```
产品: 护肤精油套装

当前定价分析:
┌──────────────────────┐
│ 我们的价格: ¥280     │
│ 市场均价:  ¥250     │
│ 竞争对手价: ¥240    │
└──────────────────────┘

差异计算:
价格差异 = ¥280 - ¥240 = ¥40
差异率 = 40 / 240 = 16.7%

结论:
优先级: 🔴 高优先级
建议: "价格偏高，建议降价"
理由: 比竞争对手价高16.7%,严重影响竞争力

预期效果:
如果降价到 ¥265:
  └─ 预期销量提升: 15-20%
  └─ 竞争力提升: 显著

新利润率估算:
成本 ¥100 → 售价 ¥265 → 利润率 165%
```
