
## ❓ 遇到问题？

### 找不到实例？
- 搜索: `ma961120`
- 或访问直接链接: https://console.cloud.tencent.com/cdb

### 添加规则时看不到"添加"按钮？
- 确认在"入站规则"部分
- 点击"编辑规则"或"新增规则"

### 添加后还是连接不上？
- 等待 2-5 分钟规则生效
- 检查 IP 是否正确: 112.117.97.242
- 检查端口是否正确: 3306
- 重启后端

### 不确定是否成功？
- 运行诊断工具
- 在前端创建一条数据并刷新
