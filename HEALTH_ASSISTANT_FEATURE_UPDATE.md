## 💡 功能说明

### 健康助手功能
```
智能拍照分析系统，包含:

1. 皮肤检测
   - 肤质诊断
   - 问题识别
   - 产品推荐

2. 美容诊断
   - 美容评分
   - 疗程方案
   - 产品推荐

3. 舌苔检测
   - 体质诊断
   - 中医调理
   - 食疗方案
```

```
✅ 为客户拍照分析
✅ 制定个性化美容方案
✅ 了解客户肤质和体质
✅ 提供专业建议
```
