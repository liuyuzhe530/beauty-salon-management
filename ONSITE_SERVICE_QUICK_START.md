## 🚀 三步快速体验

### 第1步：打开应用
```
访问 http://localhost:3000
登录系统
```

### 第2步：进入上门服务
```
点击 "客户管理" → "到店服务" 标签
```

### 第3步：立即预约
```
✅ 选择服务（6种可选）

✅ 点击"确认预约"
```

## 🎯 核心功能一览

### 📋 **立即预约** 
最直观的预约流程，5步快速完成预约

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 选择服务 | 6种热门服务任选，实时显示价格/时长 |

| ✅ 完成评价 | 服务完成后即刻评价 |

查看全部完成订单，支持评分和评价

### 👩‍💼 **美容师**
浏览全部美容师，选择最适合的专业人士

## ❓ 常见问题

### Q1: 可以取消订单吗？
```
✅ 可以！只要订单还没开始服务，就可以取消
点击"取消订单"按钮即可
支持全额退款到原支付方式
```

### Q2: 美容师离线了怎么办？
```
✅ 选择其他在线美容师即可
系统自动筛选该服务的在线美容师
离线美容师的预约按钮自动禁用
```

### Q3: 服务地址有限制吗？
```
✅ 有！服务范围包括：
- 朝阳区 ✅
- 海淀区 ✅
- 东城区 ✅
- 西城区 ✅
其他地区暂不支持
```

### Q4: 没有收到消息怎么办？
```
✅ 可以：
1. 直接拨打美容师电话
2. 在订单页面发送消息
3. 查看预计到达时间
如仍未联系上，可取消订单
```

### Q5: 可以给美容师评价吗？
```
✅ 可以！订单完成后：
1. 系统弹出评价窗口
2. 选择1-5星评分
3. 输入评价文字（可选）
4. 提交评价
您的评价对改进服务质量很重要！
```
