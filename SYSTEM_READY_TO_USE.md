##  **如何安装 MySQL (可选)**

### **Windows**
```powershell
# 方式1: 使用 Chocolatey
choco install mysql

# 方式2: 下载安装程序
# 访问 https://dev.mysql.com/downloads/mysql/
# 下载 MySQL Community Server
```

### **Mac**
```bash
brew install mysql
brew services start mysql
```

### **Linux**
```bash
sudo apt-get install mysql-server
sudo systemctl start mysql
```
