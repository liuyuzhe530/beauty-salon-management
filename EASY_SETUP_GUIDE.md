# 🚀 超简单！只需 5 个步骤

## 这是最简单的版本，跟着做就行！

---

## ✅ 第 1 步：打开腾讯云

**请您做这一个动作：**

在浏览器打开这个地址：
```
https://console.cloud.tencent.com/
```

**登录您的腾讯云账号**

---

## ✅ 第 2 步：进入 MySQL 页面

**在左侧菜单找到并点击：**

```
云产品
    ↓
搜索框输入: "MySQL"
    ↓
点击 "MySQL" 或 "云数据库 MySQL"
```

**或者直接用这个快捷链接：**
```
https://console.cloud.tencent.com/cdb
```

---

## ✅ 第 3 步：找到您的实例

**您会看到一个列表，找到这个实例：**

```
ma961120-0gn5q3yfd523d6c0
```

**点击这个实例名称**

---

## ✅ 第 4 步：进入安全组

**在实例详情页，找到顶部的标签，点击：**

```
安全组
```

**会看到 "已关联的安全组"，有个组 ID，点击这个组 ID**

---

## ✅ 第 5 步：添加规则（最重要！）

**在安全组页面，点击：**

```
编辑规则 → 添加规则
```

**填写以下内容（复制粘贴！）：**

### 协议端口
```
TCP:3306
```

### 授权对象
```
112.117.97.242/32
```

### 策略
```
选择：允许
```

### 备注
```
Local Dev
```

**点击 ✅ 确定**

---

## ✅ 第 6 步：等待 + 重启

**等待 2 分钟规则生效**

然后在您的电脑 **PowerShell** 运行：

```powershell
cd E:\xincs\xincs\backend
npm run dev
```

**看到这个信息说明成功了：**

```
✅ Database connected successfully
```

---

## 🎉 完成！

您的数据库连接成功了！

现在可以：
- ✅ 打开前端：http://localhost:5173
- ✅ 创建数据
- ✅ 刷新浏览器，数据仍然存在
- ✅ 享受完整系统！
