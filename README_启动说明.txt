【重要】请按照以下步骤操作：

1️⃣  找到这个文件所在的文件夹
   路径: E:\xincs\xincs

2️⃣  在同一个文件夹中找到:
   📄 快速启动.bat    ← 推荐用这个！(最简单)
   
3️⃣  双击运行 快速启动.bat
   
4️⃣  等待 5-10 秒，浏览器会自动打开
   
5️⃣  看到登录页面后，输入:
   用户名: admin
   密码: Admin@123
   
6️⃣  点击"登录"按钮
   
🎉 完成！系统已启动！

================================================================================

⚠️  重要提醒：

✓ 会打开 2 个黑色终端窗口，这是正常的，不要关闭它们！
✓ 第一个窗口显示后端服务器日志
✓ 第二个窗口显示前端开发日志

如果浏览器没有自动打开，手动访问: http://localhost:5173

================================================================================

【故障排查】

问题: 浏览器显示"无法连接"
解决: 等待 30 秒后按 F5 刷新页面

问题: 登录失败
检查: 确保用户名是小写 admin，密码是 Admin@123（注意大小写）

问题: 终端显示错误
查看: 查看黑色终端窗口中的错误信息
