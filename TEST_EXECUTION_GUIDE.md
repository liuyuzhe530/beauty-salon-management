**目标：** 完整测试商城装修编辑器的所有功能

##  前置准备（5分钟）

### 第 1 步：启动开发服务器

**在 PowerShell 中执行：**
```powershell
cd E:\xincs\xincs
npm run dev
```

**等待看到：**
```
  Local:   http://localhost:5173/
```

**如果出现错误：**
```
如果看到错误，可能原因：
1. Node.js 未安装 → 访问 nodejs.org 安装
2. 依赖未安装 → 执行 npm install
3. 端口被占用 → 修改配置或关闭其他应用
```

### 第 2 步：打开浏览器

**在浏览器地址栏中打开：**
```
http://localhost:5173
```

**应该看到系统登录页面。**

### 第 3 步：进入商城装修

**方式 A：底部导航**
- 点击底部导航栏的"商城装修"按钮

**方式 B：侧边栏**
- 点击侧边栏菜单
- 找到"商城"选项
- 点击"装修"

**成功标志：**
- 看到" 小程序店铺装修"标题
- 左侧显示组件库
- 右上角显示"预览页面"、"选择模板"、"管理装修"三个按钮

##  常见问题处理

### 问题 1：页面不加载
```
症状：浏览器显示空白或无法连接
解决：
1. 确认开发服务器已启动
2. 查看控制台是否有错误信息
3. 尝试 F5 刷新页面
4. 关闭浏览器重新打开
```

### 问题 2：组件不显示
```
症状：点击添加组件但编辑区不显示
解决：
1. F5 刷新页面
2. 打开浏览器控制台检查错误
3. 尝试添加其他类型组件
```

### 问题 3：编辑面板不打开
```
症状：点击组件但没有编辑面板
解决：
1. 确认点击的是组件卡片
2. 查看控制台错误
3. 刷新页面重试
```

### 问题 4：预览显示不正确
```
症状：预览模式中看不到组件
解决：
1. 确保组件已启用（复选框勾选）
2. 返回编辑器确认组件存在
3. 刷新预览页面
```
